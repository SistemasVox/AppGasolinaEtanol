# AppGasolinaEtanol
Este aplicativo foi desenvolvido com o objetivo de auxiliar motoristas e entusiastas de automóveis a calcular a proporção ideal e o custo de combustível para seus veículos. Ele oferece uma solução prática e fácil de usar para otimizar o abastecimento, considerando diferentes tipos de combustível, como gasolina e etanol, e seus respectivos preços.
